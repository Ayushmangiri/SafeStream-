# SafeStream 🛡️
AI-Powered Content Moderation Platform

## What is SafeStream?
SafeStream automatically detects toxic text and NSFW 
images using AI — keeping your feed clean and safe.

- Database: MongoDB
- AI Text: Perspective API
- AI Image: Google Vision API
- Queue: Celery + Redis
